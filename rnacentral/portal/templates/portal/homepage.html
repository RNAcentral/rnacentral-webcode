RNAcentral is a new resource to organise data for non-protein coding RNA genes

Expert Databases

  • The RNAcentral sequences are provided by a group of expert databases and supplemented by sequences from the INSDC.

  • tmRNA Website

    The tmRNA Website contains predicted tmRNA sequences from RefSeq prokaryotic genomes, plasmids and bacteriophages.
    21,318 sequences

    Examples: URS0000646B13, URS000064AECD, URS000064B0CC.

  • VEGA

    VEGA (Vertebrate Genome Annotation) is a repository for high-quality gene models produced by the manual annotation of vertebrate genomes.
    21,388 human lncRNA sequences

    Examples: URS00006398DB, URS000063A296, URS0000638AD4.

  • miRBase

    miRBase is a database of published miRNA sequences and annotations.
    3,661 sequences

    Examples: URS0000026D73, URS000052CD0D, URS00005BE7F9.

  • SRPDB

    SRPDB (Signal Recognition Particle Database) provides aligned, annotated and phylogenetically ordered sequences related to structure and function of SRP.
    855 sequences

    Examples: URS000030A37C, URS0000227674, URS000005F2FD.

Citing RNAcentral

If you use RNAcentral, please cite the following paper:

Bateman, A., Agrawal, S., Birney, E., Bruford, E. A., Bujnicki, J. M., Cochrane, G., Cole, J. R., et al. (2011). RNAcentral: A vision for an international database of RNA sequences. RNA.

About RNAcentral Learn more

The RNAcentral portal:

  • provides data access via sequence and meta-data search
  • links together the expert databases
  • provides unique identifiers for individual molecules
